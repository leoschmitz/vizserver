Informações para o uso do servidor:
===================================
Passo 1:
--------

- Para não usando muito espaço no source safe, não foram incluídas cotações
históricas da bovespa. Elas estão disponíveis no próprio site da bovespa:
www.bovespa.com.br em Mercado -> Acoes -> Dados Históricos -> Cotações Históricas 
Tem que possuir cadastro pra poder baixar.
- Depois de baixar os arquivos históricos, colocá-los no root junto dos cpps e hs.


(Para usar as ferramentas de conversão)
=======================================
Passo 2:
--------

- Buscar os códigos dos emissores mais atuais.
	- Pegar também no site da bovespa em
	mercado -> consulta -> ISIN (Banco de dados completo)
- Colocar o EMISSOR.TXT no BovespaData

Passo 3:
--------

- Buscar os títulos negociáveis do dia.
   - Buscar em mercado -> consulta -> titulos negociaveis
- Colocar TITULOS_NEGOCIAVEIS.TXT no BovespaData.
